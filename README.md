# Clinic Management System

A professional clinic management application built with Microsoft Access 2019, designed to streamline daily clinic operations.

## Key Features:
- **User Authentication**: Secure login with custom access levels (Admin/User).
- **Patient Management**: Complete record-keeping for patients.
- **Appointment Scheduling**: Integrated calendar for managing patient visits with Persian date support.
- **Financial Management**: Built-in cash register for tracking payments in Rial.
- **Reporting**: Automated generation of financial and patient reports.

## How to use:
1. Ensure Microsoft Access 2019 is installed on your system.
2. Download the `.accdb` file.
3. Open the file and log in with your credentials.
